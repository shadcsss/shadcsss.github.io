
# Shadcss

A CSS‑only port of [shadcn/ui](https://github.com/shadcn-ui/ui).

## Overview

Shadcss provides a framework‑agnostic, stylesheet‑based implementation of the shadcn/ui component library. It works directly with standard HTML, enabling you to use the design system without React or any JavaScript framework.

## Features

- **CSS Only**  
  No JavaScript frameworks or build tools required.

- **Semantic HTML**  
  Components are built using clean, accessible HTML elements.

- **Theme Support**  
  Includes light and dark mode styling.

- **Tailwind Color Palette**  
  Uses the full Tailwind CSS color system.

- **Radix Icons**  
  Compatible with the Radix Icons set.

## Usage

Include the stylesheet in your HTML:

```html
<link
  rel="stylesheet"
  href="ttps://github.com/shadcsss/shadcsss.github.io/raw/refs/heads/main/shadcss.csss"
/>
```

Build interfaces using the component classes and semantic HTML.
